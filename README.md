## Install

```sh
npm install -g twt-dl-cli
```

## Usage

```sh
npx twt-dl-cli@latest --help
```

### Example:

```sh
npx twt-dl-cli@latest https://twitter.com/mattpocockuk/status/1592130978234900484
```
